Hardware

Hardware Diagram

The diagram below shows how all of the components are physically connected and wired. One tricky part of this project was that the capacitive touch sensor outputs logic high when triggered, while the audio board waits for logic low to begin playing audio. Therefore, the sensor's output signal was inverted using a transistor and a couple of resistors, so that the devices work together. The speaker used also has it's own internal battery, so the sombrero can easily get through all of the songs without needing a recharge.
